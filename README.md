# ng2-tree

Angular 2 runtime console debugger

## Installation

```
npm install --save ng2-tree-viewer
```

## Usage

See `example/`.

```ts
import {enableNg2Tree} from "ng2tree";

if(isProd) {
  ...
} else {
  // ng2tree can be used in development mode.
  enableNg2Tree();
}
```
